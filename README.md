# patalpu-oro-kokybes-stebejimo-sistema
ESP32 pagrįsta patalpų oro kokybės stebėjimo sistema

Jungimas sensoriu prie esp32
| DHT22 | ESP32      |
| ----- | ---------- |
| VCC   | 3V3        |
| GND   | GND        |
| DATA  | GPIO4 (D4) |

| MQ-135 | ESP32  |
| ------ | ------ |
| VCC    | 5V     |
| GND    | GND    |
| A0     | GPIO34 |
